# Design Notes

## Task

Tasks should contain context info:
- Project
- Type of Work
    - Analytical
    - Creative
    - Mindless
    - ...
- Area of Focus
- Goal
- Location ("@Context" in GTD)

## Table component

Table should handle:
- Headings
- Data rows/items
- Pagination
- Sorting
- Filtering
- Expand/collapse row details
